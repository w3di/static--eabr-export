Общая информация о проекте

В рамках проекта подготовлена проектно-сметная документация (ПСД), необходимая для рассмотрения возможности финансирования Банком связанного инвестиционного проекта. ПСД готовилась с привлечением одной из крупнейших в мире инжиниринговых компаний.

Цели и задачи проекта

Подготовка ПСД по инвестиционному проекту для включения в пакет документов, необходимых для рассмотрения возможности финансирования Банком, заключения финансовых соглашений и выполнения отлагательных условий до начала финансирования Инвестиционного проекта.
